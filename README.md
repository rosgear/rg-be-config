## <img src="https://raw.githubusercontent.com/rosgear/rg-be-config/refs/heads/main/assets/images/icon.svg" width="64px" height="64px" align="absmiddle"> Модуль «Конфигурация»

Модуль входит в состав Панели управления GePanel.

Модуль предназначен для настройки и изменения конфигурации веб‑приложения и входящих в его состав компонентов, включая модули, расширения модулей, виджеты и плагины.

### Установка

Для добавления модуля в ваш проект, вы можете просто выполнить команду ниже:

```
$ composer require rosgear/rg-be-config
```

или добавить в файл composer.json вашего проекта:
```
"require": {
    "rosgear/rg-be-config": "*"
}
```
или скачать архив на [странице модуля](https://rosgear.ru/component/rg-be-config/) в каталоге приложений RosGear.

После добавления модуля в проект выполните его установку в редакцию веб‑приложения с помощью Панели управления GePanel.

## <img src="https://raw.githubusercontent.com/rosgear/rg-be-config/refs/heads/main/assets/images/icon.svg" width="64px" height="64px" align="absmiddle"> Module «Configuration»

This module is part of the GePanel Control Panel.

This module is designed for configuring and modifying the web application and its components, including modules, module extensions, widgets, and plugins.

### Installation

To add the module to your project, you can simply run the command below:

```
$ composer require rosgear/rg-be-config
```

or add to your project's composer.json file:
```
"require": {
    "rosgear/rg-be-config": "*"
}
```
or download the archive from the [module page](https://rosgear.ru/component/rg-be-config/) in the RosGear application catalog.

After adding the module to the project, install it into the web application edition using the GePanel Control Panel.
